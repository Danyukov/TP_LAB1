## Цель работы
- Ознакомиться с Git и GitHub Actions (CI).
- Получить навыки написания ООП-кода на Python и модульных тестов (pytest).
- Настроить автоматическую проверку стиля (pycodestyle) и тестирование.

## Описание проекта
Проект читает данные о студентах и их оценках из файла, рассчитывает:
- средний рейтинг каждого студента (среднее по всем дисциплинам),
- **(вариант 1)** количество студентов-отличников (у которых по всем предметам баллы ≥ 90).

Поддерживаемые форматы входных файлов:
- TXT (TextDataReader)
- YAML (YAMLDataReader) — добавлено в рамках индивидуального задания

## Используемые технологии
- Python 3.10
- pytest
- pycodestyle (PEP8)
- GitHub Actions
- PyYAML (для YAML)

## Структура проекта
- `src/` — исходный код
- `test/` — модульные тесты
- `data/` — примеры входных данных
- `.github/workflows/` — CI pipeline GitHub Actions
